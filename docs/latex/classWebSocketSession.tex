\doxysubsection{Detailed Description}
Represents an individual client Web\+Socket connection. 

Manages the lifecycle of a single Web\+Socket connection from client acceptance through message handling to disconnection. Each connected client gets its own \doxylink{classWebSocketSession}{Web\+Socket\+Session} instance.

\begin{DoxyParagraph}{Connection Lifecycle}

\begin{DoxyEnumerate}
\item Created when client connects to server
\item Web\+Socket handshake performed
\item Initial cinema data sent to client
\item Continuous message processing loop
\item Cleanup on disconnection or error
\end{DoxyEnumerate}
\end{DoxyParagraph}
\begin{DoxyParagraph}{Message Flow}

\begin{DoxyItemize}
\item Receives messages from client asynchronously
\item Forwards messages to server for processing
\item Sends responses back to client
\item Participates in broadcast messages to all clients
\end{DoxyItemize}
\end{DoxyParagraph}
\begin{DoxyParagraph}{Thread Safety}
Uses Boost.\+Beast\textquotesingle{}s async operations for thread-\/safe message handling. Write operations are queued to prevent overlapping sends. 
\end{DoxyParagraph}
