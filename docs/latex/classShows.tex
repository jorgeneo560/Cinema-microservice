\doxysubsection{Detailed Description}
Server-\/side movie show data structure with thread-\/safe seat management. 

Represents a movie show with thread-\/safe seat management.

Identical to client-\/side \doxylink{classShows}{Shows} class but used in server context. Provides thread-\/safe operations for managing seat availability and bookings.

\begin{DoxyParagraph}{Thread Safety}
All operations are thread-\/safe using shared\+\_\+mutex for concurrent access.
\end{DoxyParagraph}
\begin{DoxySeeAlso}{See also}
\doxylink{classShows}{Shows} in \doxylink{cinema__Client_8hpp}{cinema\+\_\+\+Client.\+hpp} for detailed documentation
\end{DoxySeeAlso}
This class serves as the core data structure for the Cinema booking system. It provides thread-\/safe operations for managing seat availability and bookings for individual movie shows.

\begin{DoxyParagraph}{Thread Safety}
All operations are thread-\/safe using shared\+\_\+mutex for concurrent access. Multiple readers can access seat data simultaneously, but write operations are exclusive.
\end{DoxyParagraph}
\begin{DoxyParagraph}{Seat Numbering}
Seats are numbered 1-\/20 inclusive. Internal storage uses 0-\/based indexing.
\end{DoxyParagraph}
\begin{DoxyInvariant}{Invariant}
Total seats is always 20 

Seat numbers are 1-\/20 inclusive 

seats vector size is always 20 
\end{DoxyInvariant}
