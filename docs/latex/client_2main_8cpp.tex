Main application for Cinema booking system client

This application provides the main entry point for the Cinema booking client. It integrates the Web\+Socket communication layer (\doxylink{classCinemaClient}{Cinema\+Client}) with the user interface (\doxylink{classCinemaUI}{Cinema\+UI}) to provide a complete booking experience.

\begin{DoxyParagraph}{Application Flow}

\begin{DoxyEnumerate}
\item Initialize client and UI components
\item Connect to Cinema server via Web\+Socket
\item Fetch initial cinema data from server
\item Present interactive menu to user
\item Handle user selections (view movies, book seats, help)
\item Process booking requests with server communication
\item Display booking results and confirmations
\item Graceful shutdown on exit
\end{DoxyEnumerate}
\end{DoxyParagraph}
\begin{DoxyParagraph}{Key Features}

\begin{DoxyItemize}
\item Real-\/time server communication
\item Interactive booking interface
\item Error handling and connection management
\item Booking confirmation system
\item Help and guidance for users
\end{DoxyItemize}
\end{DoxyParagraph}
\begin{DoxyParagraph}{Architecture Role}
Serves as the business logic coordinator in the client application, orchestrating communication between UI and Web\+Socket layers. 
\end{DoxyParagraph}
