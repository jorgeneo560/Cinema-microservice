\doxysubsection{Detailed Description}
Web\+Socket client for Cinema booking system. 

Provides communication layer between Cinema client application and server. Handles connection management, message sending/receiving, and protocol parsing. Maintains local cache of \doxylink{classShows}{Shows} data synchronized with server.

\begin{DoxyParagraph}{Thread Safety}
This class is thread-\/safe. All public methods can be called from multiple threads. Internal message processing runs on a separate background thread.
\end{DoxyParagraph}
\begin{DoxyParagraph}{Connection Lifecycle}

\begin{DoxyEnumerate}
\item Create \doxylink{classCinemaClient}{Cinema\+Client} instance
\item Call \doxylink{classCinemaClient_a1f89a855f15fbe3b9a321927522e7a68}{connect()} to establish Web\+Socket connection
\item Send messages via \doxylink{classCinemaClient_acb6d41c939b3a8426d64c20b730c4887}{send\+Message()}
\item Receive responses via \doxylink{classCinemaClient_a727eae2dcf7eff6b750338a87cd1627d}{get\+Last\+Response()}/get\+Last\+Booking\+Response()
\item Get synchronized \doxylink{classShows}{Shows} data via \doxylink{classCinemaClient_ae2639bd1a1bd0cabd48c7fe7bba103a5}{get\+Shows()}
\item Call \doxylink{classCinemaClient_a08575ad7810a6a5ae067d4285c258b31}{disconnect()} or destructor handles cleanup
\end{DoxyEnumerate}
\end{DoxyParagraph}
