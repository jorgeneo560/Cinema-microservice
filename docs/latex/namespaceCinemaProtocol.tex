Server-\/side protocol constants for Cinema communication. 

Protocol constants for Cinema server communication.

Defines headers, footers, and constants used in server-\/client communication.

Defines the string patterns and prefixes used in the Cinema booking protocol for parsing server responses and identifying message types.
