\doxysubsection{Detailed Description}
Service for formatting cinema data for client communication. 

Provides static methods to format \doxylink{classShows}{Shows} data into protocol-\/compliant strings for transmission to clients. Handles both initial data streams and booking update notifications.

\begin{DoxyParagraph}{Output Format}
Generates structured text output with theater grouping, movie listings, and seat availability information in a human-\/readable format. 
\end{DoxyParagraph}
